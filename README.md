# Degrees
此问题要求给出任意两个影星，给出他们之间的联系(通过电影)

这个问题主要难点在于两点：
1. 抽象表示图和边
2. 搜索算法

其中:
1. 重点在于理解电影id，影星id
2. 利用BFS就可以，也没啥启发式搜索，不好优化
